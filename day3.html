DAY 3

HTML

  • HTML - Hyper-text Markup Language
  • HTML was meant for documents instead of application layouts.

Document Flow

  • absolute
  • fixed
  • relative

Display

  • Block - You can change height and width, but nothing else will be on the same line. It takes up the entire width.
  • Inline - Cannot change height and width, but other things can be next to it.
  • Inline-block - Height and width can be adjusted, plus other things can sit next to it.

Important things to note

  • margin: auto - Centers div
  • Boilerplate - The stuff that's always there.
